## Install

with curl

```bash
curl -fsSL --proto '=https' --tlsv1.2 \
  https://raw.githubusercontent.com/aklozyp/Moddex/main/scripts/bootstrap.sh \
  | bash
```

or with wget

```bash
wget -qO- --https-only --secure-protocol=TLSv1_2 \
  https://raw.githubusercontent.com/aklozyp/Moddex/main/scripts/bootstrap.sh \
  | bash
```
